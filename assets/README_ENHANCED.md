# TG Signer 增强版 WebUI

## 🎉 新功能概览

这是TG Signer的增强版本，提供了完整的前端化操作体验和强大的任务管理功能。

### ✨ 核心改进

1. **前端化登录系统**
   - 完全在Web界面完成账号登录
   - 支持用户名+密码或验证码登录
   - 安全的会话管理和状态保持

2. **实时任务监控**
   - 完整的任务生命周期管理
   - 实时状态显示（运行中/就绪/错误）
   - 一键启动、停止、测试

3. **增强用户界面**
   - 直观的仪表板设计
   - 任务总览和快速操作
   - 搜索和过滤功能

4. **后台服务架构**
   - 配置文件自动监控
   - 资源优化管理
   - 错误处理和日志记录

## 🎯 功能使用指南

### 第一步：账号登录

1. **进入登录界面**：启动后会自动跳转到登录页面
2. **输入凭据**：
   - 用户名：Telegram用户名或手机号
   - 密码：您的Telegram密码
   - 验证码：可选（如果启用了两步验证）
3. **点击登录**：🚀 "开始登录"按钮
4. **等待验证**：系统会验证凭据并创建会话

### 第二步：主界面导航

登录成功后，您将看到以下标签页：

| 标签 | 功能 | 说明 |
|------|------|------|
| 📊 总览 | 任务总览面板 | 显示所有任务的状态统计 |
| ⚙️ 配置管理 | 配置编辑器 | 创建和管理签到配置 |
| 👤 用户信息 | 个人信息 | 查看账户和对话信息 |
| 📋 签到记录 | 历史记录 | 查看所有签到历史 |
| 📝 日志 | 系统日志 | 查看运行日志 |
| 🎯 任务管理 | 任务控制 | 集中管理所有任务 |

### 第三步：任务管理

#### 查看任务状态
- **🟢 就绪**：配置完整，可以启动
- **🔴 运行中**：正在执行签到任务
- **🟡 配置错误**：需要修复配置问题
- **⚪ 未配置**：还没有创建配置

#### 操作任务
- **🧪 测试**：验证配置是否正确
- **▶️ 开始**：启动签到任务
- **⏹️ 停止**：停止运行中的任务
- **🔄 刷新**：更新任务状态

#### 批量操作
- **🔄 刷新所有任务**：更新所有任务状态
- **🧪 测试所有配置**：批量验证配置
- **⏸️ 暂停所有任务**：暂停运行中的任务
- **🛑 停止所有任务**：停止所有运行中的任务

### 第四步：配置管理

#### 创建新配置
1. 切换到"配置管理"标签页
2. 选择"签到配置"子标签
3. 点击"交互式配置"按钮
4. 按照向导步骤完成配置
5. 保存配置

#### 编辑现有配置
1. 从任务列表中选择要编辑的任务
2. 点击"编辑"按钮
3. 修改配置参数
4. 保存更改

#### 测试配置
1. 选择要测试的任务
2. 点击"🧪 测试"按钮
3. 查看测试结果
4. 根据结果调整配置

## 🔧 技术细节

### 架构设计
```
前端 (NiceGUI)
├── 认证系统 (WebAuthSystem)
├── 任务监控 (TaskMonitor)
├── UI组件 (enhanced_app.py)
└── 数据管理 (data.py)

后端 (Python)
├── Telegram API 集成
├── 任务调度引擎
├── 数据库管理
└── 日志系统
```

### 状态管理
- **前端状态**: 浏览器localStorage
- **后端状态**: SQLite数据库 + 内存缓存
- **任务状态**: 实时同步到前端

### 性能优化
- **懒加载**: 按需加载大列表数据
- **缓存机制**: 减少重复API调用
- **异步处理**: 非阻塞UI更新

## 🐛 故障排除

### 常见问题

#### Q: 登录失败
A: 检查以下项目：
- 确保用户名和密码正确
- 检查网络连接
- 确认API密钥已设置

```bash
export TG_API_ID=your_api_id
export TG_API_HASH=your_api_hash
```

#### Q: 任务不运行
A: 检查配置是否正确：
- 确认至少有一个聊天配置
- 检查签到时间设置
- 验证动作配置

#### Q: WebUI界面卡顿
A: 尝试以下方法：
- 清理浏览器缓存
- 关闭不必要的标签页
- 检查系统资源使用情况

### 错误代码参考

| 错误码 | 含义 | 解决方案 |
|--------|------|----------|
| AUTH_ERROR | 认证失败 | 重新登录 |
| CONFIG_ERROR | 配置错误 | 检查配置文件 |
| NETWORK_ERROR | 网络问题 | 检查网络连接 |
| TIMEOUT_ERROR | 超时 | 增加超时设置 |

### 系统要求
- **CPU**: 1GHz以上
